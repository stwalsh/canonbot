Colophon

The Thinkatron Review is a hand-picked selection from the daily output of Lucubrator, a Claude instance that reads the English poetry canon. Production is collaborative across several Claude instances: one writes the pieces, one handles editorial polish, one worked on design and build. A human sits between them, choosing.

Typography

Text face
EB Garamond throughout — Octavio Pardo’s Latin-extended continuation of Georg Duffner’s OFL-licensed revival, served as a self-hosted variable font (weight axis 400–800, roman and italic). Real small caps and old-style figures.
Fallback
Iowan Old Style, Charter, Palatino, Georgia.
Accent
Oxblood in light mode, terracotta in dark. Mode follows system preference.
Ornament
The floral heart (U+2766, or hedera) at the masthead rule and closing each piece is drawn from EB Garamond’s own glyph set.

Pipeline

Text
Generated by Lucubrator, a retrieval-augmented Claude instance reading the English poetry canon.
Editorial
A daily model pass identifies candidates for keeping; the editor hand-picks from that cut. A second Claude instance — ‘the sub’ — handles line editing and register consistency when run against a piece.
Design
Developed in conversation with another Claude instance across successive sessions, against a brief of print-derived literary-journal typography.
Build
Static site generated from SQLite via Jinja2; a quote parser in the build step handles inline italicised quotation and block insets. Deployed to Netlify from GitHub.
